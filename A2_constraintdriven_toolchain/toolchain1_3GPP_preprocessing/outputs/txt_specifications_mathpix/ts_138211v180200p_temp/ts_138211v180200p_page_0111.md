The cyclic shift \( \alpha_{i} \) for antenna port \( p_{i} \) is given as
\[
\alpha_{i}=2 \pi\left(\frac{n_{\mathrm{SRS}}^{\mathrm{cs}, i}}{n_{\mathrm{SRS}}^{\mathrm{cs}, \mathrm{max}}}+\frac{f_{\mathrm{csh}}\left(n_{\mathrm{f}}, n_{\mathrm{s}, \mathrm{f}^{\prime}}^{\mu}, l^{\prime}\right)}{K n_{\mathrm{SRS}}^{\mathrm{cs} \max }}\right)
\]
where
\[
n_{\mathrm{SRS}}^{\mathrm{cs}, i}=\left\{\begin{array}{ll}
\left(n_{\mathrm{SRS}}^{\mathrm{cs}}+\frac{n_{\mathrm{SRS}}^{\mathrm{cs} \max }\left\lfloor\left(\bar{p}_{i}-1000\right) / 4\right\rfloor}{\bar{N}_{\mathrm{ap}}^{\mathrm{SRS}} / 4}\right) \bmod n_{\mathrm{SRS}}^{\mathrm{cs} \max } & \text { if } \bar{N}_{\mathrm{ap}}^{\mathrm{SRS}}=8 \text { and } n_{\mathrm{SRS}}^{\mathrm{cs}, \max }=6 \\
\left(n_{\mathrm{SRS}}^{\mathrm{cs}}+\frac{n_{\mathrm{SRS}}^{\mathrm{cs} \max }\left\lfloor\left(\bar{p}_{i}-1000\right) / 2\right\rfloor}{\bar{N}_{\mathrm{ap}}^{\mathrm{SRS}} / 2}\right) \bmod n_{\mathrm{SRS}}^{\mathrm{cs} \max } & \text { if } \bar{N}_{\mathrm{ap}}^{\mathrm{SRS}}=4 \text { and } n_{\mathrm{SRS}}^{\mathrm{cs}, \max }=6 ; \text { or if } \bar{N}_{\mathrm{ap}}^{\mathrm{SRS}}=8 \text { and } n_{\mathrm{SRS}}^{\mathrm{cs} \max }=12 \\
\left(n_{\mathrm{SRS}}^{\mathrm{cs}}+\frac{n_{\mathrm{SRS}}^{\mathrm{cs} \max }\left(\bar{p}_{i}-1000\right)}{\bar{N}_{\mathrm{ap}}^{\mathrm{SRS}}}\right) \bmod n_{\mathrm{SRS}}^{\mathrm{cs}, \max } & \text { otherwise }
\end{array}\right.
\]
where \( n_{\mathrm{SRS}}^{\mathrm{cs}} \in\left\{0,1, \ldots, n_{\mathrm{SRS}}^{\mathrm{cs} \text {, max }}-1\right\} \) is contained in the higher layer parameter transmission Comb. The maximum number of cyclic shifts \( n_{\text {SRS }}^{\text {cs, } \max } \) is given by Table 6.4.1.4.2-1.

The quantities \( \bar{p}_{i} \) and \( \bar{N}_{\text {ap }}^{\text {SRS }} \) are given by
- if the higher-layer parameter nrofSRS-Ports-n8 equals ports8tdm
\[
\begin{aligned}
\bar{p}_{i} & =\left\{\begin{array}{ll}
1000+p_{i} \bmod 2 & \text { if } p_{i}-1000<4 \\

\end{array}\right. \\
\bar{N}_{\mathrm{ap}}^{\mathrm{SRS}} & =4
\end{aligned}
\]
- otherwise
\[
\begin{aligned}
\bar{p}_{i} & =p_{i} \\
\bar{N}_{\mathrm{ap}}^{\mathrm{SRS}} & =N_{\mathrm{ap}}^{\mathrm{SRS}}
\end{aligned}
\]
