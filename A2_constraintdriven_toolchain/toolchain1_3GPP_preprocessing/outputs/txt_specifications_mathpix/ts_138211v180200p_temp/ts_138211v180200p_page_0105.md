For the purpose of PT-RS mapping, the resource blocks allocated for PUSCH transmission are numbered from 0 to \( N_{\text {RB }}-1 \) from the lowest scheduled resource block to the highest. The corresponding subcarriers in this set of resource blocks are numbered in increasing order starting from the lowest frequency from 0 to \( N_{s c}{ }^{\mathbf{R B}} N_{\mathbf{R B}}-1 \). The subcarriers to which the PT-RS shall be mapped are given by
\[
\begin{aligned}
k & =k_{\mathrm{ref}}^{\mathrm{RE}}+\left(i K_{\mathrm{PT}-\mathrm{RS}}+k_{\mathrm{ref}}^{\mathrm{RB}}\right) N_{\mathrm{sc}}^{\mathrm{RB}} \\
k_{\mathrm{ref}}^{\mathrm{RB}} & =\left\{\begin{array}{ll}
n_{\mathrm{RNTI}} \bmod K_{\mathrm{PT}-\mathrm{RS}} & \text { if } N_{\mathrm{RB}} \bmod K_{\mathrm{PT}-\mathrm{RS}}=0 \\
n_{\mathrm{RNTI}} \bmod \left(N_{\mathrm{RB}} \bmod K_{\mathrm{PT}-\mathrm{RS}}\right) & \text { otherwise }
\end{array}\right.
\end{aligned}
\]
where
- \( \quad i=0,1,2 \ldots \)
- \( \quad k_{\text {ref }}^{R E} \) is given by Table 6.4.1.2.2.1-1 for the DM-RS port associated with the PT-RS port according to clause 6.2.3 in [6, TS 38.214]. If the higher-layer parameter resourceElementOffset in PTRS-UplinkConfig is not configured, the column corresponding to 'offset00' shall be used.
- \( n_{\text {RNTI }} \) is the RNTI associated with the DCI scheduling the transmission using C-RNTI, CS-RNTI, MCS-CRNTI, SP-CSI-RNTI, or is the CS-RNTI in case of configured grant
- \( \quad N_{\text {RB }} \) is the number of resource blocks scheduled
- \( \quad K_{\text {PT-RS }} \in\{2,4\} \) is given by \( [6, \mathrm{TS} 38.214] \).
