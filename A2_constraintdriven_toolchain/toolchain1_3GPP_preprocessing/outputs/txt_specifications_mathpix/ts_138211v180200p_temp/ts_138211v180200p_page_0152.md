8.3.1.2 Modulation

For the single codeword \( q=0 \), the block of scrambled bits shall be modulated, resulting in a block of complex-valued modulation symbols \( d^{(q)}(0), \ldots, d^{(q)}\left(M_{\text {symb }}^{(q)}-1\right) \) where \( M_{\text {symb }}^{(q)}=M_{\text {symb }, 1}^{(q)}+M_{\text {symb }, 2}^{(q)} \).

Modulation for \( 0 \leq i<M_{\mathrm{bit}, \text { SCI2 }}^{(q)} \) shall be done as described in clause 5.1 using QPSK, where \( M_{\text {symb, } 1}^{(q)}=M_{\mathrm{bit}_{1}, \text { SCl2 }}^{(q)} / 2 \).
Modulation for \( M_{\mathrm{bit}, \mathrm{SCI} 2}^{(q)} \leq i<M_{\mathrm{bit}}^{(q)} \) shall be done as described in clause 5.1 using one of the modulation schemes in Table 8.3.1.2-1 where \( M_{\text {symb, } 2}^{(q)}=M_{\text {bit, data }}^{(q)} / Q_{\mathrm{m} \text { - }} \)
