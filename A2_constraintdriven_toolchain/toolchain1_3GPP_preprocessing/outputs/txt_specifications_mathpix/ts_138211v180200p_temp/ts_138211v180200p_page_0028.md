5.3.1 OFDM baseband signal generation for all channels except PRACH and RIM-RS

The time-continuous signal \( s_{\ell}^{(p, \mu)}(t) \) on antenna port \( p \) and subcarrier spacing configuration \( \mu \) for OFDM symbol \( l \in\left\{0,1, \ldots, N_{\text {slot }}^{\text {subrame, } \mu} N_{\text {symb }}^{\text {slet }}-1\right\} \) in a subframe for any physical channel or signal except PRACH is defined by
\[
\begin{array}{l} 
s_{l}^{(p, \mu)}(t)=\left\{\begin{array}{ll}
\bar{s}_{l}^{(p, \mu)}(t) & t_{\text {start }, l}^{\mu} \leq t<t_{\text {start }, l}^{\mu}+T_{\text {symb }, l}^{\mu} \\
0 & \text { otherwise }
\end{array}\right. \\
\bar{s}_{l}^{(p, \mu)}(t)=\sum_{k=0}^{N_{\text {grid }, x}^{\text {size }, \mu} N_{\mathrm{sc}}^{\mathrm{RB}}-1} a_{k, l}^{(p, \mu)} e^{j 2 \pi\left(k+k_{0}^{\mu}-N_{\mathrm{grid}, x}^{\text {size }, \mu} N_{\mathrm{sc}}^{\mathrm{RB}} / 2\right) \Delta f\left(t-N_{\mathrm{Cr}, l}^{\mu} T_{\mathrm{c}}-t_{\mathrm{start}, l}^{\mu}\right)} \\
k_{0}^{\mu}=\left(N_{\mathrm{grid}, x}^{\text {start }, \mu}+N_{\mathrm{grid}, x}^{\text {size }, \mu} / 2\right) N_{\mathrm{sc}}^{\mathrm{RB}}-\left(N_{\mathrm{grid}, x}^{\text {start }, \mu_{0}}+N_{\mathrm{grid}, x}^{\text {size }, \mu_{0}} / 2\right) N_{\mathrm{sc}}^{\mathrm{RB}} 2^{\mu_{0}-\mu} \\
T_{\mathrm{symb}, l}^{\mu}=\left(N_{\mathrm{u}}^{\mu}+N_{\mathrm{CP}, l}^{\mu}\right) T_{\mathrm{c}}
\end{array}
\]
