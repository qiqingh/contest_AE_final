7.4.1.1.2 Mapping to physical resources

The UE shall assume the PDSCH DM-RS being mapped to physical resources according to configuration type 1 or configuration type 2 as given by the higher-layer parameter dmrs-Type.

The UE shall assume the sequence \( r(m) \) is scaled by a factor \( \beta_{\text {PDSCH }}^{\text {DMRS }} \) to conform with the transmission power specified in [6, TS 38.214] and mapped to resource elements ( \( k, l)_{p, \mu} \) according to
- if the higher-layer parameter dmrs-TypeEnh is configured
\[
\begin{array}{l} 
\tilde{a}_{k, l}^{\left(p_{j, l}\right)}=\beta_{\mathrm{PDSCH}}^{\mathrm{DMRS}} w_{\mathrm{f}}\left(k^{\prime}\right) w_{\mathrm{t}}\left(l^{\prime}\right) r\left(4 n+k^{\prime}\right) \\
k=\left\{\begin{array}{ll}
8 n+2 k^{\prime}+\Delta & \text { configuration type 1 } \\
12 n+k^{\prime}+\Delta & \text { configuration type 2, } k^{\prime}=0,1 \\
12 n+k^{\prime}+\Delta+4 & \text { configuration type 2, } k^{\prime}=2,3
\end{array}\right. \\
k^{\prime}=0,1,2,3 \\
l=\bar{l}+l^{\prime} \\
n=0,1, \ldots \\
j=0,1, \ldots, v-1
\end{array}
\]
- otherwise
\[
\tilde{a}_{k, l}^{\left(p_{j}, \mu\right)}=\beta_{\mathrm{PDSCH}}^{\mathrm{DMRS}} w_{\mathrm{f}}\left(k^{\prime}\right) w_{\mathrm{t}}\left(l^{\prime}\right) r\left(2 n+k^{\prime}\right)
\]
