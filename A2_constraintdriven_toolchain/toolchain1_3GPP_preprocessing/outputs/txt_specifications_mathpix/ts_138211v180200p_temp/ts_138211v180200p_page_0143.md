7.4.1.6.4.5 Mapping between resource triplet and set ID

The resource indices \( i_{\mathrm{t}}^{\mathrm{RIM}}, i_{\mathrm{f}}^{\mathrm{RIM}} \), and \( i_{\mathrm{s}}^{\mathrm{RIM}} \) are determined from the index \( \bar{r} \) in the set ID \( n_{\text {setID }} \) according to
\[
\begin{array}{l}
i_{\mathrm{t}}^{\mathrm{RIM}}=T_{\text {start }}+\left(\left\lfloor\frac{n_{\text {setID }}}{N_{\mathrm{s}}^{\mathrm{RIM}}}\right\rfloor \bmod N_{\mathrm{t}}^{\mathrm{RIM}}\right) R_{i}+\bar{r} \\
i_{\mathrm{f}}^{\mathrm{RIM}}=\left(\left\lfloor\frac{n_{\text {setID }}}{N_{\mathrm{t}}^{\mathrm{RIM}} N_{\mathrm{s}}^{\mathrm{RIM}}}\right\rfloor \bmod N_{\mathrm{f}}^{\mathrm{RIM}}\right) \\
i_{\mathrm{s}}^{\mathrm{RIM}}=S_{\text {start }}+\left(n_{\text {setID }} \bmod N_{\mathrm{s}}^{\mathrm{RIM}}\right)
\end{array}
\]
where
- \( \quad N_{\mathrm{t}}{ }^{\mathrm{RIM}} \) is given by
\[
N_{\mathrm{t}}^{\mathrm{RIM}}=\left\{\begin{array}{ll}
\left\lceil\frac{N_{\text {setl }}^{\mathrm{RIM}}}{N_{\mathrm{f}}^{\mathrm{RIM}} N_{\mathrm{s}}^{\mathrm{RIM}, 1}}\right\rceil & \text { for RIM-RS type } 1 \text { and if EnoughIndication is disabled } \\
{\left[\frac{2 N_{\mathrm{setID}}^{\mathrm{RIM}, 1}}{N_{\mathrm{f}}^{\mathrm{RIM}} N_{\mathrm{s}}^{\mathrm{RIM}, 1}}\right]} & \text { for RIM-RS type } 1 \text { and if EnoughIndication is enabled } \\
{\left[\frac{N_{\mathrm{setI} \mathrm{D}}^{\mathrm{RII}, 2}}{N_{\mathrm{f}}^{\mathrm{RIM}} N_{\mathrm{s}}^{\mathrm{RIM}, 2}}\right]} & \text { for RIM-RS type } 2
\end{array}\right.
\]
- \( \quad N_{\mathrm{f}}{ }^{\mathrm{RIM}} \in\{1,2,4\} \) is the number of candidate frequency resources configured in the network;
- \( \quad N_{\mathrm{s}}^{\mathrm{RIM}} \) is the number of sequence candidates for the current RIM-RS resource given by
\[
N_{\mathrm{s}}^{\mathrm{RIM}}=\left\{\begin{array}{cl}
N_{\mathrm{s}}^{\mathrm{RIM}, 1} & \text { for RIM-RS type } 1 \text { and if EnoughIndication is disabled } \\
N_{\mathrm{s}}^{\mathrm{RIM}, 1} / 2 & \text { for RIM-RS type } 1 \text { and if EnoughIndication is enabled } \\
N_{\mathrm{s}}^{\mathrm{RIM}, 2} & \text { for RIM-RS type } 2
\end{array}\right.
\]

\[
